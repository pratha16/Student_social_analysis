# Student_social_analysis
# 📊 Data Analysis Project (Python)

## 📌 Project Overview
This project performs **data preprocessing and exploratory data analysis (EDA)** on a dataset using Python.  
The goal is to clean raw data, uncover patterns, and extract meaningful insights through visualization.

---

## 🎯 Objectives
- Clean and preprocess raw data
- Handle missing and inconsistent values
- Perform exploratory data analysis (EDA)
- Visualize trends and relationships
- Generate actionable insights

---

## 🛠️ Tech Stack
- **Python**
- **Pandas** – data manipulation
- **NumPy** – numerical operations
- **Matplotlib & Seaborn** – data visualization
- **Jupyter Notebook**

---

## 📂 Project Structure

├── data.csv # Raw dataset
├── data_preprocessing.ipynb # Data cleaning & preprocessing
├── eda.ipynb # Exploratory Data Analysis
└── README.md # Project documentation


---

## ⚙️ Data Preprocessing
Key steps performed:
- Handling missing values (removal/imputation)
- Removing duplicates
- Data type conversions
- Feature cleaning and formatting

---

## 📈 Exploratory Data Analysis (EDA)
Analysis includes:
- Distribution of variables
- Correlation analysis
- Trend identification
- Data visualization using graphs and plots

---

## 📊 Key Insights
- Identified important patterns and trends in the dataset  
- Found relationships between variables  
- Highlighted potential areas for further analysis or modeling
